h1. Introduction

twitter-list-stats is a Web app to to monitor members and subscribers of public Twitter lists on a daily time scale.

It's R&D project to explore and practice the following languages, tools and methods:

* Python
* Google Application Engine
* Twitter API
* Vagrant and Chef
* TDD
